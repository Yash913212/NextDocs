# Commencer avec NextDocs

Ce guide vous aidera à démarrer avec NextDocs en quelques minutes.

## Prérequis

Avant de commencer, assurez-vous que vous avez:

- Node.js 18 ou supérieur
- npm ou gestionnaire de paquets yarn
- Un éditeur de code (VS Code recommandé)
- Une compréhension de base de Next.js

## Installation

### Configuration Locale

1. Clonez le référentiel:

```bash
git clone https://github.com/yourusername/nextdocs.git
cd nextdocs
```

2. Installez les dépendances:

```bash
npm install
```

3. Démarrez le serveur de développement:

```bash
npm run dev
```

4. Ouvrez votre navigateur et naviguez vers:

```
http://localhost:3000
```
