# Introduction

Bienvenue sur **NextDocs**, un portail de documentation moderne construit avec Next.js.

## Qu'est-ce que NextDocs?

NextDocs est une plateforme de documentation complète conçue pour livrer du contenu à une audience mondiale.

## Commencer

Pour commencer avec NextDocs:

1. **Explorez la Documentation**: Utilisez la barre latérale pour parcourir les différents sujets
2. **Recherchez**: Utilisez la barre de recherche pour trouver des informations spécifiques
3. **Essayez les Exemples**: Exécutez les exemples de code fournis dans la documentation
4. **Fournissez des Commentaires**: Partagez vos impressions en utilisant le widget de commentaires

## Caractéristiques Principales

### Support Multilingue

NextDocs supporte plusieurs langues:
- Anglais (en)
- Espagnol (es)
- Français (fr)
- Allemand (de)

### Recherche Plein Texte

Utilisez la barre de recherche pour trouver du contenu dans toute la documentation.

### Blocs de Code

Tous les exemples de code incluent un bouton de copie:
- Cliquez sur le bouton pour copier le code
- Collez dans votre éditeur
- Exécutez et modifiez au besoin

## Exemple de Code

Voici un exemple simple:

```python
def bonjour_monde():
    print("Bonjour, NextDocs!")

bonjour_monde()
```
