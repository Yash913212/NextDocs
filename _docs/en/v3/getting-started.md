# Getting Started with NextDocs v3

The latest version of NextDocs with cutting-edge features.

## What's New

### v3 Features

- AI-powered search
- Advanced analytics
- Collaboration features
- Improved performance

### Installation

```bash
npm install
npm run dev
```

### Configuration

Edit `.env` to configure your instance:

```env
NEXT_PUBLIC_SITE_NAME=MyDocs
NEXT_PUBLIC_SITE_URL=https://docs.example.com
```
