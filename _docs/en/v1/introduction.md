# Introduction

Welcome to **NextDocs**, a modern, high-performance documentation portal built with Next.js.

## What is NextDocs?

NextDocs is a comprehensive documentation platform designed to deliver content to a global audience. It features:

- **Multi-Language Support**: Available in English, Spanish, French, and German
- **Fast Performance**: Statically generated pages with Incremental Static Regeneration
- **Easy Navigation**: Intuitive sidebar and table of contents
- **Full-Text Search**: Search across all documentation
- **Code Examples**: Easy-to-copy code blocks
- **API Documentation**: Interactive Swagger UI
- **User Feedback**: Collect feedback directly from users

## Getting Started

To get started with NextDocs, follow these steps:

1. **Explore the Documentation**: Use the sidebar to browse different topics
2. **Search**: Use the search bar to find specific information
3. **Try Examples**: Run code examples provided in the documentation
4. **Provide Feedback**: Let us know your thoughts using the feedback widget

## Key Features

### 1. Multiple Versions

Access documentation for different versions of the product:
- Version 1: Original documentation
- Version 2: Updated features
- Version 3: Latest features

Use the version selector to switch between versions.

### 2. Internationalization

NextDocs supports multiple languages:
- English (en)
- Spanish (es)
- French (fr)
- German (de)

Use the language switcher in the header to change languages.

### 3. Table of Contents

Each page includes an auto-generated table of contents that:
- Links to all headings on the page
- Highlights the current section as you scroll
- Provides quick navigation

### 4. Search Functionality

Use the search bar to find content across all documentation:
- Search by keyword
- Get instant results
- Click results to navigate

### 5. Code Blocks

All code examples include a copy button:
- Click the button to copy code
- Paste into your editor
- Run and modify as needed
