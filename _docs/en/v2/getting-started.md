# Getting Started with NextDocs v2

Version 2 has been redesigned from the ground up with performance and user experience in mind.

## Key Improvements

### Performance

- 50% faster page loads
- Optimized images
- Reduced bundle size

### User Experience

- Simplified navigation
- Improved search
- Better accessibility

## Installation

```bash
npm install
npm run dev
```

## Features

### Full-Text Search

Enhanced search now includes:
- Fuzzy matching
- Faceted search
- Search suggestions

### Version Management

Easily switch between v1, v2, and v3 documentation.

### Multi-Language Support

Support for:
- English
- Spanish
- French
- German
