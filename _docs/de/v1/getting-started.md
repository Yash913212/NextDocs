# Erste Schritte mit NextDocs

Diese Anleitung hilft Ihnen, in wenigen Minuten mit NextDocs zu beginnen.

## Voraussetzungen

Bevor Sie beginnen, stellen Sie sicher, dass Sie haben:

- Node.js 18 oder höher
- npm oder yarn Paketmanager
- Ein Code-Editor (VS Code empfohlen)
- Grundverständnis von Next.js

## Installation

### Lokale Einrichtung

1. Klonen Sie das Repository:

```bash
git clone https://github.com/yourusername/nextdocs.git
cd nextdocs
```

2. Installieren Sie die Abhängigkeiten:

```bash
npm install
```

3. Starten Sie den Entwicklungsserver:

```bash
npm run dev
```

4. Öffnen Sie Ihren Browser und navigieren Sie zu:

```
http://localhost:3000
```

## Projektstruktur

```
nextdocs/
├── app/              # Next.js App-Verzeichnis
├── components/       # Wiederverwendbare Komponenten
├── _docs/           # Dokumentationsinhalte
├── lib/             # Dienstprogramme und Helfer
├── public/          # Statische Ressourcen
└── ...
```
