# Einführung

Willkommen bei **NextDocs**, einem modernen Dokumentationsportal, das mit Next.js erstellt wurde.

## Was ist NextDocs?

NextDocs ist eine umfassende Dokumentationsplattform für globale Zielgruppen.

## Erste Schritte

Zum Einstieg in NextDocs:

1. **Dokumentation erkunden**: Verwenden Sie die Seitenleiste zum Durchsuchen verschiedener Themen
2. **Suchen**: Verwenden Sie die Suchleiste, um spezifische Informationen zu finden
3. **Beispiele testen**: Führen Sie in der Dokumentation bereitgestellte Codebeispiele aus
4. **Feedback geben**: Teilen Sie Ihre Gedanken mithilfe des Feedback-Widgets

## Hauptmerkmale

### Mehrsprachige Unterstützung

NextDocs unterstützt mehrere Sprachen:
- Englisch (en)
- Spanisch (es)
- Französisch (fr)
- Deutsch (de)

### Volltextsuche

Verwenden Sie die Suchleiste, um Inhalte in der gesamten Dokumentation zu finden.

### Codeblöcke

Alle Codebeispiele enthalten eine Kopierschaltfläche:
- Klicken Sie auf die Schaltfläche, um den Code zu kopieren
- Fügen Sie in Ihren Editor ein
- Führen Sie aus und ändern Sie nach Bedarf

## Codebeispiel

Hier ist ein einfaches Beispiel:

def hallo_welt():
