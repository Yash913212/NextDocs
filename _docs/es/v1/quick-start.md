# Inicio Rápido

Pon en marcha tu portal de documentación en solo unos minutos.

## Configuración Básica

### Paso 1: Crear Tu Primer Documento

Agrega un nuevo archivo Markdown en el directorio `_docs`:

```markdown
---
title: Mi Primera Página
description: Esta es mi primera página de documentación
---

# Mi Primera Página

¡Hola, NextDocs!
```

### Paso 2: Agregar Ejemplos de Código

Incluye bloques de código con resaltado de sintaxis:

```javascript
function saludar(nombre) {
  console.log(`¡Hola, ${nombre}!`)
  return `Bienvenido a NextDocs`
}

saludar('Desarrollador')
```

### Paso 3: Organizar Contenido

Estructura tu documentación con encabezados:

## Tema Principal

### Subtema 1

Contenido para subtema 1

### Subtema 2

Contenido para subtema 2
