# Preguntas Frecuentes

## Preguntas Generales

### ¿Qué es NextDocs?

NextDocs es un portal de documentación moderno construido con Next.js que proporciona soporte rápido y multilingüe para documentación técnica.

### ¿Quién debe usar NextDocs?

NextDocs es ideal para:
- Proyectos de software que requieren documentación integral
- Equipos que atienden audiencias globales
- Organizaciones que necesitan documentación de referencia de API
- Empresas con múltiples versiones de productos

### ¿Es NextDocs de código abierto?

Sí, NextDocs es de código abierto y está disponible en GitHub bajo la licencia MIT.

## Preguntas Técnicas

### ¿Qué stack tecnológico utiliza NextDocs?

NextDocs está construido con:
- **Next.js 14**: Marco moderno de React
- **Tailwind CSS**: Marco CSS first
- **TypeScript**: JavaScript type-safe
- **next-i18next**: Internacionalización
- **Zustand**: Gestión de estado
- **React Markdown**: Renderización de Markdown

### ¿Cómo funciona ISR?

ISR permite que las páginas de documentación sean:
1. Pre-construidas en tiempo de compilación
2. Regeneradas automáticamente en segundo plano cada 60 segundos
3. Servidas desde el caché mientras se generan actualizaciones

## Internacionalización (i18n)

### ¿Cuántos idiomas se admiten?

NextDocs viene preconfigurado para 4 idiomas:
- Inglés (en)
- Español (es)
- Francés (fr)
- Alemán (de)

### ¿Cómo traduzco las cadenas de la interfaz de usuario?

Las traducciones de la interfaz se almacenan en `public/locales/{locale}/common.json`.
