# Introducción

Bienvenido a **NextDocs**, un moderno portal de documentación construido con Next.js.

## ¿Qué es NextDocs?

NextDocs es una plataforma de documentación integral diseñada para entregar contenido a una audiencia global. Incluye características como:

- **Soporte Multilingüe**: Disponible en inglés, español, francés y alemán
- **Rendimiento Rápido**: Páginas generadas estáticamente con Regeneración Estática Incremental
- **Navegación Fácil**: Barra lateral intuitiva e índice de contenidos
- **Búsqueda de Texto Completo**: Busca en toda la documentación
- **Ejemplos de Código**: Bloques de código fáciles de copiar
- **Documentación de API**: Interfaz interactiva de Swagger
- **Retroalimentación del Usuario**: Recopila comentarios directamente de los usuarios

## Comenzar

Para comenzar con NextDocs, sigue estos pasos:

1. **Explora la Documentación**: Utiliza la barra lateral para examinar diferentes temas
2. **Busca**: Usa la barra de búsqueda para encontrar información específica
3. **Prueba Ejemplos**: Ejecuta ejemplos de código proporcionados en la documentación
4. **Proporciona Comentarios**: Cuéntanos tus opiniones usando el widget de comentarios

## Características Clave

### 1. Múltiples Versiones

Accede a la documentación para diferentes versiones del producto:
- Versión 1: Documentación original
- Versión 2: Características actualizadas
- Versión 3: Características más recientes

Usa el selector de versión para cambiar entre versiones.

### 2. Internacionalización

NextDocs admite múltiples idiomas:
- Inglés (en)
- Español (es)
- Francés (fr)
- Alemán (de)

Usa el selector de idioma en el encabezado para cambiar idiomas.

### 3. Tabla de Contenidos

Cada página incluye una tabla de contenidos generada automáticamente que:
- Enlaza a todos los encabezados de la página
- Destaca la sección actual mientras te desplazas
- Proporciona navegación rápida

### 4. Funcionalidad de Búsqueda

Usa la barra de búsqueda para encontrar contenido en toda la documentación:
- Búsqueda por palabra clave
- Obtén resultados instantáneos
- Haz clic en los resultados para navegar

### 5. Bloques de Código

Todos los ejemplos de código incluyen un botón de copia:
- Haz clic en el botón para copiar el código
- Pega en tu editor
- Ejecuta y modifica según sea necesario
